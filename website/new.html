The BlueBox Project

The BlueBox is a compact digital library that provides preloaded educational content to support learning in remote areas. The materials are available offline and can be accessed for free by any device that connects to the BlueBox hotspot. Whether through a wireless or wired connection, communities, educators, and students can access the content using any device's web browser, ensuring access to valuable educational resources even in places with no internet connectivity. Below are small projects that I developed to enhance the Project.

Shapes Quiz

Shape Quiz module helps students learn about 2D and 3D shapes and their key characteristics through interactive questions. Dr. Ormond made final modifications to ensure alignment with other modules in the BlueBox project.

Camera Stream

The external camera setup, based on EbenKouao's GitHub source code, was significantly modified to enable live streaming via a Flask server. It allows students to participate in real-time activities like experiments or crafts.

Internal Camera

The Text Editor includes an internal webcam for recording and uploading videos offline, enabling educators and healthcare professionals to create multimedia-rich documents for lessons, presentations, or patient education in remote areas.

Text Editor

This Text Editor enables users to create, upload, and edit files using the provided toolbar and save files ending with .html. It also offers auto-save every time users make changes in the Text Editor box.

Code Editor

This Code Editor enables users to create, upload, rename, delete, and save .html files, with auto-update and auto-save ensuring changes are preserved. It helps students learn basic coding concepts interactively.

CCI Financial Trakcer

This financial tracker, created for Chronic Care International (CCI) under the BlueBox Project, helps non-profit businesses manage transactions and analyze performance.
