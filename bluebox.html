The BlueBox Project

The BlueBox is a compact digital library that provides preloaded educational content to support learning in remote areas. Accessible offline by any device connected to the hotspot, it ensures communities and students can continue learning even without internet. Below are interactive modules I developed to enhance the BlueBox experience.

Shapes Quiz

Interactive module to learn 2D & 3D shapes through quizzes.

Camera Stream

Attached a separeted camera for real-time experiments and in-class activities.

Internal Camera

Webcam-enabled text editor for recording lessons offline.

Text Editor

An offline text editor with auto-save for continuous offline works.

Code Editor

Interactive coding practice tool with file management.

CCI Financial Tracker

Custom financial tracker to manage finances and performance for nonprofit business.
